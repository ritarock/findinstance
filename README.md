# findinstance
## install
```
$ git clone https://github.com/ritarock/findinstance.git
$ cd findinstance/
$ go install
```

## Usage
```
NAME:
   findinstance - Find ec2 instance

USAGE:
    [global options] command [command options] [arguments...]

COMMANDS:
   help, h  Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --profile value  set aws profile
   --help, -h       show help
```
